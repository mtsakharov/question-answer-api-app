## TheQuestions - Laravel Questions/Answers App Rest Api

## Features

- Laravel 8
- Laravel Sanctum
- Docker
- Tests

## Installation
- Clone the repo
- Edit `.env` and set your database connection details
- Run `composer intall`
- Create docker image by running `./vendor/bin/sail up`
- `sail artisan migrate` run migrations
